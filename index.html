Gaji stores every company, staff record and payslip in a Google Sheet you own — free, private, and easy to back up. One-time setup, about 3 minutes.

  1. Open Google Sheets, create a blank sheet, then go to Extensions → Apps Script.
  2. Delete the placeholder code, paste in the gaji-sheets-backend.gs script provided, and save.
  3. Click Deploy → New deployment → Web app. Set Execute as: Me and Who has access: Anyone, then deploy and authorize.
  4. Copy the Web App URL it gives you and paste it below.
